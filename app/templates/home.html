Teste Funcional

Este é um teste de unidade funcional que verifica se o título da página inicial contém a mensagem "Prof: Rennê Santos"..

O teste usa a biblioteca Selenium para simular a interação do usuário com o navegador e garantir que a página é carregada corretamente.

Em seguida, ele verifica se a mensagem "Prof: Rennê Santos" está presente no título da página usando a asserção assertIn().

Esse tipo de teste é comum no TDD (Desenvolvimento Orientado a Testes), que é uma metodologia de desenvolvimento de software que prioriza a escrita de testes antes da implementação do código.

O objetivo é garantir que o software atenda aos requisitos do usuário e minimize a ocorrência de bugs e erros.
